The simplest way to add Google analytics to your site with no bloat.

== Description ==

This plugin simply inserts the Google analytics tracking code in the <head></head> tags of your site with no bloat. All you have to do is paste in your tracking code, click save and you're done.

= Compatibility =

Tested up to Wordpress 4.9.1


== Installation ==

1. Copy the `super-simple-google-analytics` folder into your `wp-content/plugins` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste your Google analytics tracking code into the box and press save
4. Done!
